Install Instructions: 
1) Install node for javascript.
2) Install the npm modules (mpm init). 
3) Navigate to the working directory (using cd) to run the server file. 
4) Use 'node server.js' to run the server, once in the right directory. 
5) In any browser, once server is running, paste 'http://localhost:3000/chatClient.html' to see the html webpage.
6) You can open multiple webpages and they will update in real time using socket.io
7) Create valid users and chat and use direct message features that will be displayed.
8) The code will also implement the css styling according to the different type of messages.
